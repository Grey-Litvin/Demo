Embeding .avi files in a web page

 

Here is the code:

-------Start cut below here.

<OBJECT CLASSID="clsid:22d6f312-b0f6-11d0-94ab-0080c74c7e95F"
HEIGHT=150
WIDTH=150
NAME=Msshow1
ID=Msshow1>
<PARAM NAME="FileName" VALUE="fireworks.AVI">
<PARAM NAME="autoStart" VALUE="true">
<PARAM NAME="showControls" VALUE="false">
<PARAM NAME="PlayCount" VALUE="20">
<embed src="fireworks.AVI" width="150" height="150" controller="false" autostart="true" loop=true>
</EMBED>
</OBJECT>

-------End Cut above here.

The only thing you need to change is The Name of the File and the Height and Width of the window to display it in.
